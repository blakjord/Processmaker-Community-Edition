Steps

The Steps option shows all Dynaforms, Output and Input Documents assigned to the current task, allowing each document to be easily accessed.

The Steps option is available when starting a case from the New Case option or when opening the case from the Inbox, Draft, Review or Reassign trays.

Note: Take into account the following notes:

  • To access the case from the previously mentioned trays, the user must be assigned to the case.
  • Cases in the Review tray have the Steps option that just displays Dynaforms assigned to the supervisor. For more information about the Review tray, see Process Supervisor.
  • Cases in the Reassign tray displays the Steps option if the case is assigned to the supervisor. For more information about the Reassign tray, see Process Supervisor.

Assigning Tasks

When all steps in a task have been completed, the case is ready to be routed to the next task. At this point, a window will display the next task's information.

Where:

  • Next Task: Name of the next task where the case will be routed to.
  • Employee: Name of the user who will be in charge of executing the task.
